# keelim-maestro Subproject Intros Design

## Style Prompt

Workspace Atlas: a precise Swiss editorial system for a multi-repo engineering workspace. The video should feel like a fast, source-led project map rather than a marketing reel. Use visible grid structure, strong type, restrained motion, and a few operational accent colors to distinguish project lanes.

## Colors

- Paper background: `#f4f1e8`
- Ink foreground: `#161616`
- Panel surface: `#fffaf0`
- Primary accent: `#0b63ff`
- Secondary accent: `#f2a900`
- Support accents: `#0a7f5a`, `#c74332`

## Typography

- Primary UI and headline stack: `Apple SD Gothic Neo`, `Noto Sans KR`, `Helvetica Neue`, Arial, sans-serif.
- Headlines use 800-900 weight. Supporting copy uses 700-800 weight for video legibility.

## Motion

- Grid-locked entrances, short vertical reveals, and staggered operational blocks.
- No random motion. No floating decorative blobs. No infinite loops.
- Each project gets a clear 5-7 second chapter with the same rhythm so the viewer can compare projects quickly.

## What NOT to Do

- Do not blur child-repo boundaries into a single monorepo narrative.
- Do not use generic neon SaaS gradients.
- Do not add fake screenshots or unsupported product claims.
- Do not introduce audio or voiceover unless the script is reviewed separately.
